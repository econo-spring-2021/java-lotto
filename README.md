# java-lotto
로또 시뮬레이터 미션을 진행하는 저장소

## 구현할 기능 목록

* [x] 필요한 domain 생성
* [x] 구입금액 입력
* [x] 로또 생성
    * [x] 개수 계산 및 출력
    * [x] 로또 번호 생성
    * [x] 로또 번호 출력 
* [x] 당첨 번호, 보너스 볼 입력
* [ ] 당첨 확인
    * [x] 당첨 결과 계산
    * [x] 당첨 결과 출력
      * [x] 누락사항 리팩토링
    * [x] 수익률 계산 및 출력
* [x] 입출력 예외 캐치
* [x] 테스트 코드 작성
